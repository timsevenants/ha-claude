# ha-claude add-ons

Eigen, zelf-onderhouden Home Assistant add-on repository met **Claude Terminal**: een
web-terminal met de [Claude Code](https://github.com/anthropics/claude-code) CLI erin.

Dit is een eigen rebuild, geïnspireerd op het (niet langer onderhouden) origineel van
`heytcass/home-assistant-addons`, bewust minimaal en gemakkelijk te onderhouden gehouden.

## Add-ons

- **[Claude Terminal](./claude-terminal)** — web-terminal met Claude Code CLI en persistente login.

## Installeren

1. **Instellingen → Add-ons → Add-on Store**
2. Drie-puntjes-menu rechtsboven → **Repositories**
3. Voeg deze repo-URL toe en klik **Toevoegen**
4. Installeer **Claude Terminal**

Zie [claude-terminal/DOCS.md](./claude-terminal/DOCS.md) voor details.
